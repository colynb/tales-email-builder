Issue #22 - Oct 16, 2014

By Colyn Brown under issues

Greetings! Welcome to issue #22 of Tales from the Frontend. I sorted through quite bit this week, it was tough to keep it at a reasonable size. I've included a couple items to help pave the way for ES6 Harmony the upcoming sixth major release of the ECMAScript language specification. Enjoy your weekend!

Skill building...

Understanding ECMAScript 6
NICHOLAS C. ZAKAS' complete guide for transitioning between ECMAScript 5 and 6. Not specific to any JavaScript environment, so it is equally useful to web developers as it is Node.js developers.

leanpub.com javascript es6

Express Routing - The Beginners Guide
A succinct guide to creating routes in Express 4.0

jilles.me javascript expressjs

2005 Guest Lecture by Mark Zuckerberg [Video]
On 7 December 2005, an unpolished Mark Zuckerberg guest lectured a CS class to discuss how Facebook works. You have to watch this if you're interested in app and db scaling.

youtube.com scale mysql php

Animating SVG with CSS
Always the maverick, in this walk-thru, CHRIS COYIER explains an alternative animation method for SVG.

css-tricks.com css svg animation

Tutorial: Build your own Google TV Using RaspberryPi, NodeJS and Socket.io
Be a hardware hacker! A fun proof of concept for building your own Google TV complete with remote app!

codementor.io raspberrypi nodejs socketio

An Introduction To The HTML5 History API
With the HTML5 History API, you can manipulate the browsers history and change urls without refreshing the page. Learn how this API came to be, then build a single-page app to demonstrate it.

tutsplus.com html5 javascript

How to Write Highly Scalable and Maintainable JavaScript: Coupling
ALEX CASTROUNIS guides us through his 4th and final chapter in writing scalable javascript apps, with approaches to reduce coupling between modules, keeping them from getting too cozy.

innoarchitech.com javascript

Preparing for ECMAScript 6: Set and WeakSet
You better start getting on the ES6 train. In this post, AURELIO DE ROSA walks through a couple new data types in ES6 that can be quite useful.

sitepoint.com javascript es6

koajs – Your first application using the KAMN stack – part 1/2
Koa is a new web framework designed by the team behind Express, which aims to be a smaller, more expressive, and more robust foundation for web applications and APIs. MICHEL ALBERS shows you how to write a basic web application with a REST api with this new framework.

tastenjesus.de nodejs koa

CSS-Only Solution For UI Tracking
KRASIMIR TSONEV describes a pretty nifty CSS-only approach for tracking UI interactions using Google Analytics.

smashingmagazine.com analytics css

Node Punishes Developers Because it Disobeys the Unix Way
A bonehead's 2011 unintentionally hilarious rant against nodejs and nodejs devs.

semitwist.com nodejs

Node-Webkit File Player – A File player for your desktop
Arvind Ravulavaru built a nice little file browser using Node-Webkit a runtime based on Chromium for writing native apps in HTML and Javascript.

thejackalofjavascript.com nodejs

Is AngularJS Fast Enough?
For many React devs, there is no debate. If you care about performance, you use React instead of Angular. However, as JEFF WHELPLEY explains, it's not so simple and there are ways to improve Angular's percieved performance.

medium.com javascript angularjs reactjs
